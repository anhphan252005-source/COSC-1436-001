# Lake Palestine Fishing Game
This project is a simple C++ fishing game set at Lake Palestine. The player chooses a fish they want to catch and then chooses a bait. The goal is to choose the correct bait for the selected fish. The fishing trip is based on the idea of catching fish for a charity food organization.

The game includes four fish choices: Bass, Catfish, Bluegill, and Crappie. It also includes six bait choices: Nightcrawlers, Shad bait fish, Live minnow, Plastic worms, French fries, and Pickles. Each useful bait only catches one specific fish. French fries and pickles do not catch any fish and are used as funny bait choices.

At the end of each round, the game displays a final result showing the fish selected, whether the fish was caught, and the total number of failure attempts. If the player catches the fish, the game displays a champion message.

Overall, the purpose of this game is to practice beginner C++ in a fun and interactive way. The game uses choices, menus, messages, and results to help show how C++ can be used to create a simple real-life themed program. It also introduces players to different fish and bait options while keeping the game easy to understand and enjoyable to play.
